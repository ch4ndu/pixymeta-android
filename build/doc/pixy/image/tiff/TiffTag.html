GRAY_RESPONSE_UNIT

public static final TiffTag GRAY_RESPONSE_UNIT
The precision of the information contained in the GrayResponseCurve. Because optical density is specified in terms of fractional numbers, this field is necessary to interpret the stored integer information. For example, if GrayScaleResponseUnits is set to 4 (ten-thousandths of a unit), and a GrayScaleResponseCurve number for gray level 4 is 3455, then the resulting actual value is 0.3455.
